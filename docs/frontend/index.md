# 前端开发

Web 前端技术体系，从基础三件套到现代框架和工程化实践。

## 目录

- [HTML/CSS](./html-css.md) — 语义化标签、布局、响应式、动画
- [JavaScript](./javascript.md) — ES6+、异步编程、闭包、原型链
- [TypeScript](./typescript.md) — 类型系统、泛型、工具类型
- [React](./react.md) — 组件、Hooks、状态管理、Next.js
- [Vue](./vue.md) — 响应式、组合式 API、Vite、Nuxt
- [工程化](./tooling.md) — 构建工具、测试、CI/CD、性能优化

## 学习路线建议

1. HTML + CSS 基础 → 能还原设计稿
2. JavaScript 深入 → 理解语言本质
3. 选一个主流框架（React 或 Vue）深入
4. 学习工程化工具链和性能优化
