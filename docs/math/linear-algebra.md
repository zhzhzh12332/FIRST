# 线性代数

线性代数是机器学习、计算机图形学和数据科学的数学基础。

## 向量

向量是既有大小又有方向的量，可以理解为 n 维空间中的一个点。

### 基本运算

```python
import numpy as np

# 向量定义
v = np.array([1, 2, 3])
w = np.array([4, 5, 6])

v + w      # [5, 7, 9]      — 加法
v - w      # [-3, -3, -3]    — 减法
2 * v      # [2, 4, 6]       — 标量乘法
v @ w      # 32              — 点积 (1*4 + 2*5 + 3*6)
np.cross(v, w)  # [-3, 6, -3] — 叉积
```

### 范数

| 范数 | 公式 | 含义 |
|------|------|------|
| L1 范数 | $\|x\|_1 = \sum\|x_i\|$ | 曼哈顿距离 |
| L2 范数 | $\|x\|_2 = \sqrt{\sum x_i^2}$ | 欧几里得距离（常用）|
| 无穷范数 | $\|x\|_\infty = \max\|x_i\|$ | 最大分量 |

::: tip
L2 范数是机器学习中最常用的距离度量。L1 范数有稀疏性，常用于 Lasso 回归。
:::

## 矩阵

### 基本运算

```python
# 矩阵运算
A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

A @ B        # 矩阵乘法
A.T          # 转置
np.linalg.inv(A)    # 逆矩阵
np.linalg.det(A)    # 行列式
```

**矩阵乘法规则：**
```
A(m×n) @ B(n×p) = C(m×p)
要求 A 的列数等于 B 的行数
```

### 特殊矩阵

| 矩阵类型 | 性质 | 应用 |
|---------|------|------|
| 单位矩阵 $I$ | 对角线为 1，其余为 0 | 矩阵乘法的单位元 |
| 对角矩阵 | 非对角线元素为 0 | 缩放变换 |
| 对称矩阵 | $A = A^T$ | 协方差矩阵 |
| 正交矩阵 | $Q^T Q = I$ | 旋转、PCA |
| 正定矩阵 | $x^T A x > 0$ | 优化、牛顿法 |

## 特征值与特征向量

**核心方程：**
$$A v = \lambda v$$

其中 $v$ 是特征向量，$\lambda$ 是特征值。特征向量在 $A$ 变换下只被缩放，不改变方向。

```python
A = np.array([[4, -2], [1, 1]])
eigenvalues, eigenvectors = np.linalg.eig(A)

print(eigenvalues)    # [3, 2]
print(eigenvectors)   # 对应的特征向量
```

### 特征分解
$$A = Q \Lambda Q^{-1}$$

其中 $Q$ 是特征向量矩阵，$\Lambda$ 是对角矩阵（特征值）。

### 奇异值分解 (SVD)
$$A = U \Sigma V^T$$

**SVD 是最重要的矩阵分解之一：**
- **PCA (降维)**：使用 SVD 分解协方差矩阵
- **推荐系统**：隐语义模型（SVD 分解评分矩阵）
- **图像压缩**：保留前 k 个奇异值
- **矩阵求逆**：伪逆计算

```python
# SVD 分解
U, S, Vt = np.linalg.svd(A, full_matrices=False)
```
