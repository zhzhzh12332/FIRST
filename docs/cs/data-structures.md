# 数据结构

## 线性结构

### 数组
- 连续内存空间，O(1) 随机访问
- 插入/删除 O(n)

### 链表
- 非连续存储，通过指针连接
- 插入/删除 O(1)，查找 O(n)
- 变体：双向链表、循环链表

## 树形结构

### 二叉树
- 每个节点最多两个子节点
- 遍历：前序、中序、后序、层序

### 二叉搜索树 (BST)
- 左 < 根 < 右
- 平均 O(log n)，最坏 O(n)

### 平衡树
- AVL 树、红黑树
- 通过旋转维持平衡

### 堆
- 完全二叉树，用于优先队列
- 大顶堆 / 小顶堆

## 图结构

- 有向图 / 无向图
- 邻接矩阵 / 邻接表
- 遍历：DFS、BFS

## 哈希表

- 键值映射，平均 O(1)
- 冲突解决：链地址法、开放寻址法
