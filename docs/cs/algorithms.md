# 算法

## 排序算法

| 算法 | 平均时间 | 最坏时间 | 空间 | 稳定 |
|------|---------|---------|------|------|
| 冒泡 | O(n²) | O(n²) | O(1) | 是 |
| 快排 | O(n log n) | O(n²) | O(log n) | 否 |
| 归并 | O(n log n) | O(n log n) | O(n) | 是 |
| 堆排 | O(n log n) | O(n log n) | O(1) | 否 |

## 搜索算法

- 二分查找：O(log n)，要求有序
- BFS：最短路径，队列实现
- DFS：回溯，栈/递归实现

## 算法思想

- **动态规划**：最优子结构 + 重叠子问题
- **贪心**：局部最优 → 全局最优
- **分治**：分解 → 解决 → 合并
- **回溯**：试错，不行就回退
