# 计算机基础

计算机科学核心知识体系，涵盖从数据结构到系统设计的基础内容。

## 目录

- [数据结构](./data-structures.md) — 数组、链表、树、图、哈希表等
- [算法](./algorithms.md) — 排序、搜索、动态规划、贪心等
- [操作系统](./os.md) — 进程、内存、文件系统、并发
- [计算机网络](./networking.md) — TCP/IP、HTTP、DNS、网络安全
- [数据库](./database.md) — SQL、索引、事务、NoSQL

## 学习路线建议

1. 先掌握一门编程语言（推荐 Python 或 C）
2. 学习数据结构与算法，配合 LeetCode 练习
3. 理解操作系统和网络的基本原理
4. 深入学习数据库的存储与查询优化
