# 计算机基础

计算机科学核心知识体系，涵盖从数据结构到系统设计的基础内容。

::: tip 学习建议
先掌握一门编程语言（推荐 Python 或 C），然后按以下路线循序渐进。
:::

## 目录

| 主题 | 核心内容 | 难度 |
|------|---------|------|
| [数据结构](./data-structures) | 数组、链表、树、图、哈希表 | ⭐⭐ |
| [算法](./algorithms) | 排序、搜索、动态规划、贪心 | ⭐⭐⭐ |
| [操作系统](./os) | 进程、内存、文件系统、并发 | ⭐⭐⭐ |
| [计算机网络](./networking) | TCP/IP、HTTP、DNS、网络安全 | ⭐⭐ |
| [数据库](./database) | SQL、索引、事务、NoSQL | ⭐⭐ |

## 学习路线

1. **编程语言基础** — 掌握 Python 或 C
2. **数据结构与算法** — 配合 LeetCode 练习
3. **操作系统与网络** — 理解计算机底层原理
4. **数据库** — 存储与查询优化
